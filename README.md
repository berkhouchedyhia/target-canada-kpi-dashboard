
# target-canada-kpi-dashboard
Dashboard interactif développé pour analyser la performance du système d’information de Target Canada à travers 16 KPI.  Il permet de simuler des dérives, visualiser les impacts sur les opérations et la satisfaction client, et soutenir la prise de décision grâce à des indicateurs dynamiques et des fonctionnalités interactives.

## Overview
This project presents an **interactive dashboard** developed as part of a case study on **Target Canada**.

The dashboard allows users to monitor **16 Key Performance Indicators (KPI)** related to the performance of an Information System (IS), and demonstrates how data-driven insights can support decision-making and prevent operational risks.

---

## Objectives
The main objectives of this project are:
- Monitor system performance using KPI
- Detect anomalies and operational risks
- Simulate real-world failure scenarios
- Support decision-making with data visualization
- Demonstrate a preventive approach to IT governance

---

## Key Features

- **Interactive KPI Dashboard**
  - 16 KPI distributed across 4 strategic axes:
    - Scalability of the Information System
    - Customer Satisfaction
    - Local Adaptation
    - Decision-Making Responsiveness

- **Dynamic Filtering**
  - Filter KPI by status:
    - 🔴 Critical
    - 🟠 Warning
    - 🟢 OK

- **Real-Time Interaction**
  - Clickable KPI cards
  - Dynamic data updates
  - Interactive visual feedback

- **Visual Enhancements**
  - Smooth animations and hover effects
  - Clean and modern UI
  - Responsive layout

---

## 🛠️ Technologies Used

- **HTML** → Structure of the dashboard  
- **CSS** → Design, layout, animations and visual effects  
- **JavaScript** → Interactivity and dynamic behavior  
- **Chart.js** → Data visualization

## ▶️ Running the Project Locally

### Recommended Method (VS Code)

1. Clone or download the repository
2. Open the project folder in **Visual Studio Code**
3. Install the extension **Live Server**
4. Right-click on `index.html`
5. Click **"Open with Live Server"**

The dashboard will be available at: http://127.0.0.1:5500/index.html
